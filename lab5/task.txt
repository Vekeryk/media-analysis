Лабораторна робота 5. Робота з медіафайлами JPEG

Потрібно реалізувати Python скрипт для:
1. перевірки валідності JPEG файлу,
2. отримати метадані Exif
3. Виявити фронтальні проекції облич людей на рисунку (виділити їх червоними рамками)

Для реалізації скрипта використовуєм бібліотеки opencv-python та Pillow

Для виявлення облич застосовуєм вбудований каскад Хаара (за бажанням можна використати DNN моделі).

Запуск скрипта:
python <назва скрипта>.py new_york.jpeg

Результат:
1. Файл у форматі jpg з обведеними рамками обличчями.
2. Файл у форматі json із метаданими.
3. py файл з кодом скрипта
